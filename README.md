Program jest prostą przeglądarką zdjęć o formatach BMP, JPG, PNG, GIF
Program po uruchomieniu weryfikuje, czy plik istnieje oraz czy ścieżka istnieje
W oknie głównym oprócz wyświetlanego przycisku znajdują się przyciski dodaj, wyczyść oraz szczegóły
Przycisk dodaj pozwala na podgląd zdjęcia, wyczyść usuwa zdjęcie z podglądu a szczegóły pozwalają na zobaczenie podstawowych informacji o pliku
